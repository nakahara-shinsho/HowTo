1.  [https://www.anaconda.com/download]からwindows版をダウンロードして、インストールしてください。
2.  添付のdev.yamlを使って、anacondaのVirtual Env [img-dev]を作ってください。
  - Anaconda Prompt開く
  - conda env create -f dev.yml 
4.  Virtual Envに入る
　- conda env activate img-dev
5. Jupyterを立ち上がる
　- jupyter-lab
6. ブラウザ上のIDE環境を開く
　- localhost:8888/lab
